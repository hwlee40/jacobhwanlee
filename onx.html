Typeface and specimen web page created during Marie Otsuka’s web type class. Designing key weights of OnX by hand and letting the program generate rest of the weights along two axes, I explore the different moments that arise from interpolating between two polar forms: a circle and a star.

Composed of a single component that interpolates between a circle and a star, OnX Variable is a modular font about inbetweens.

OnX Variable has two axes—weight & sharpness—and a near infinite amonut of variations.

The typeface specimen website uses a two column grid structure to mirror the two key weight system of the typeface.
